vmd controller 끄기(advanced-main에서 ctrl+s 눌러 히든옵션 띄우면 됨)
display mode : auto select

    * optimus = igpu 직결, dgpu 도움
    * dgpu only = dgpu 직결
    * auto select = nvidia 드라이버가 mux를 조정해 위 두개 중에 자동선택
